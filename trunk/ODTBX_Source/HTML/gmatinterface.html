GMAT Interface

ODTBX includes an interface to the General Mission Analysis Tool, a NASA Goddard-developed, open-source, mission design and analysis tool. This interface allows ODTBX to use GMAT instead of JAT or r2bp for force model derivatives, but currently has no utility features like coordinate system or time system conversion.

To start using the interface, follow these steps:

  1. Make a GMAT script file that configures your spacecraft initial state and force model. An example of such a file is included in the ODTBX installation at the following path: <ODTBX>/gmat/matlab/libCInterface/GmatConfig.script, where <ODTBX> is the location of your ODTBX installation.
  2. Include startgmat(<scriptfile>, <propagator>) in your ODTBX script, above the first usage of gmatforces_km. Replace <scriptfile> with the path to the GMAT script you created in step 1, and <propagator> with the name of the Propagator in the GMAT script that you want to use. You can omit this second argument if there is only one Propagator.
  3. Use gmatforces_km as your dynfun.
  4. Call closegmat() at the end of your ODTBX script.


Known limitations:

  • The GMAT interface currently does not work with drag forces, as the necessary partials are not calculated.

GMAT_Adapters
  GMAT_Adapters
 
  ODTBX Functions
 
    gmatforces_km            - Returns derivatives of GMAT Force models (state in km).
    validate_gmatforces      - Validates that the GMAT interface is working correctly.

  GMAT Functions

    closegmat                - Closes the GMAT interface.
    findodemodel             - Returns the GMAT model id for the named propagator (ODE model).
    opengmat                 - Opens the GMAT interface without loading a script or a propagator.
    preparescript            - Loads a script into the GMAT interface.
    setodemodel              - Sets the active GMAT propagator (ODE model).
    startgmat                - Starts the GMAT interface (including loading a script and a propagator).

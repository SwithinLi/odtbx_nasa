ODTBX System Requirements


You have installed the Orbit Determination Toolbox. The toolbox was created, compiled and tested using the following versions of MATLAB, JAT and Java:

1. MATLAB versions 7.9 (R2009b) and 7.11 (R2010b)
2. JAT revision 261 (ODTBX branch), Apr. 27, 2011
3. Java JDK, SDK and JRE version 1.5 or greater

If you would like to submit more code for the ODTBX project, send email to odtbx@lists.nasa.gov.

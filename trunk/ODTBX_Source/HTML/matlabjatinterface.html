MATLAB JAT Interface

When you installed the Orbit Determination Toolbox your startup.m file was edited to set the path to all JAT classes in the interface. No special user action is required to use the MATLAB JAT Interface.

Note that the MATLAB JAT Interface does not require any of the standard JAT external jar dependencies so these external jars have not been included in the installer. If someone requires these external jars then please contact the ODTBX team.

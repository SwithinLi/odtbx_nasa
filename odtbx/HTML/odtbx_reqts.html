ODTBX System Requirements


You have installed the Orbit Determination Toolbox. The toolbox was created, compiled and tested using the following operating systems and versions of MATLAB, JAT and Java:

1. Windows 7, MacOS X 10.9, Ubuntu 14.04
2. MATLAB version R2015a. Earlier versions may work, but without guarantee and there are known incompatibilities with R2011b and earlier.
3. JAT from ODTBX commit d44a587, August 26, 2012
4. Java JDK, SDK and JRE version 1.7 or greater

While other setups may partially work, they are in no way tested or supported. If you would like to submit more code for the ODTBX project, please send email to odtbx@lists.nasa.gov.

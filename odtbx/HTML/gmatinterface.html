GMAT Interface

ODTBX includes an interface to the General Mission Analysis Tool, a NASA Goddard-developed, open-source, mission design and analysis tool. This interface allows ODTBX to use GMAT instead of JAT or r2bp for force model derivatives, but currently has no utility features like coordinate system or time system conversion.

To start using the interface, follow the instructions on the ODTBX GMAT Wiki.

Known limitations:

  • The interface can include drag forces in the state derivatives (if configured), but cannot include them in the Jacobian, as the necessary information is not calculated within GMAT. [GMAT Bug #2557]
  • On 64-bit Mac, to use Solar radiation pressure, set SolarSystem.EphemerisSource = 'SPICE' in your GMAT script. Note that this will prevent outer planets (Jupiter and beyond) from being used as gravity sources. [GMAT Bug #2517]
